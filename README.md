# Kaskada website
This is the Kaskada website built with jekyll + ViteJS + VueJS

## Requirements
- Ruby ^2.7.0
- Bundler ^2.4.6
- Node ^16.17.0

## Installation
- Install the dependencies node dependencies
``` bash
npm install
```
- Install the gems
``` bash
bundle install
```

## Development

Run the script
```
bundle exec jekyll serve --livereload --open-url --drafts
```
