Open source feature engine for real-time ML

Build and operate real-time ML models directly from event data at scale.

Get Started Read the docs

Kasakada is a time-aware query abstraction, a high-performance feature computation engine, and a configurable execution model supporting bulk and real-time data processing.

ML practitioners can unify historical and real-time data processing to instantly reconstruct the historical results of a query and then deploy the same query as a live materialization without losing historical context.

You can reduce thousands of lines of SQL and avoid duplication of work across model development and deployment to deploy real-time ML models in days instead of months.
